# 極簡部落格（僅新增與刪除）

本專案已清空原有內容，僅保留最基本的兩項操作：

- 新增文章
- 刪除文章

所有資料存放於專案根目錄的 `posts.json`。

## 使用方式

1. 安裝依賴（首次執行）
   ```bash
   npm install
   ```
2. 新增文章
   ```bash
   node script.js add "文章標題" "文章內容"
   ```
3. 刪除文章
   ```bash
   node script.js delete <id>
   ```

每篇文章都會自動取得遞增的 `id` 與建立時間（ISO 格式）。命令列提示與錯誤訊息已全部改為繁體中文。
