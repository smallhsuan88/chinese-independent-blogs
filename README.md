# 極簡部落格（僅新增與刪除）

本專案已清空其他功能，只保留新增與刪除文章兩個動作，並提供圖形介面與命令列兩種方式。

## 圖形介面使用方式

1. 直接開啟 `index.html`（雙擊或用瀏覽器開啟）。
2. 按下「新增文章」按鈕，輸入標題與內容後儲存。
3. 刪除按鈕可移除指定文章。

> 文章資料會存放在瀏覽器的 localStorage（僅本機可見）。

## 命令列使用方式

1. 安裝依賴（首次執行）
   ```bash
   npm install
   ```
2. 新增文章
   ```bash
   node script.js add "文章標題" "文章內容"
   ```
3. 刪除文章
   ```bash
   node script.js delete <id>
   ```

每篇文章都會自動取得遞增的 `id` 與建立時間。命令列提示與錯誤訊息為繁體中文。
